Horizon Theme for Bootstrap



Build responsive, mobile-first projects on the web with the world's most popular Horizon Theme front-end component library.

Horizon Theme for Bootstrap is an open source toolkit based on Bootstrap for developing Horizon Theme apps with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.
